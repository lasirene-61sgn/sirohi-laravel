ADMIN PANEL API TESTING STEPS

=== AUTHENTICATION ===
1. Login to get token:
   Method: POST
   URL: http://127.0.0.1:8000/api/admin/login
   Fields:
   - email (required)
   - password (required)
   
   Response: Copy the "token" value for other requests

=== BANNER CRUD TEST ===
1. Create Banner:
   Method: POST
   URL: http://127.0.0.1:8000/api/admin/banners
   Headers: Authorization: Bearer YOUR_TOKEN
   Fields:
   - image_path (required)
   - status (required) - values: active, inactive

2. Get All Banners:
   Method: GET
   URL: http://127.0.0.1:8000/api/admin/banners
   Headers: Authorization: Bearer YOUR_TOKEN

3. Get Single Banner:
   Method: GET
   URL: http://127.0.0.1:8000/api/admin/banners/{ID}
   Headers: Authorization: Bearer YOUR_TOKEN

4. Update Banner:
   Method: PUT
   URL: http://127.0.0.1:8000/api/admin/banners/{ID}
   Headers: Authorization: Bearer YOUR_TOKEN
   Fields:
   - image_path (optional)
   - status (optional) - values: active, inactive

5. Delete Banner:
   Method: DELETE
   URL: http://127.0.0.1:8000/api/admin/banners/{ID}
   Headers: Authorization: Bearer YOUR_TOKEN
